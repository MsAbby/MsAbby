git clone

1. git clone 并上传

1.1 相关命令操作

1.1.1 命令

  • 1、将所有文件添加到版本库
    $ git add .
  • 2、将文件提交到仓库：
    $ git commit -m '提交说明'
  • 3、删除之前关联的地址：
    $ git remote rm origin
  • 4、关联到新的远程地址：
    $ git remote add origin 新的git地址（origin 是新仓库的别名）
  • 5、查看是否关联成功：
    $ git remote -v
  • 6、push 到新的分支：
    $ git push origin master
