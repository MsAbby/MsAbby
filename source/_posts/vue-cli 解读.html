1.1 config

1.1.1 publicPath

  • 1、Type: string
  • 2、Default: '/'
  • 3、webpack.dev.conf.js配置：
    publicPath: config.dev.assetsPublicPath,
    和 webpack.base.conf.js 的 output.publicPath 保持一致
  • 4、打包路径：

    默认情况下，会假设是被部署在一个域名的根路径上，例如 https://www.my-app.com/，则设置绝对路径

    publicPath : '/'

    如果被部署在一个子路径上，例如被部署在 https://www.my-app.com/my-app/，则设置相对路径

    publicPath : /my-app/ 或 '' 或 './'

  • publicPath以下情况不建议使用相对路径
  • ● 当使用基于 HTML5 history.pushState 的路由时
  • ● 当使用 pages 选项构建多页面应用时
